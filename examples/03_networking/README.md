# 🌐 Bölüm 3: Ağ ve Trafik Yönetimi Örnekleri

Bu dizin, Kubernetes içerisindeki servis iletişimi ve dış dünyaya erişim (North-South traffic) yapılandırmalarını içerir.

## 📄 Dosyalar

| Dosya | Açıklama |
|:---|:---|
| service.yaml | ClusterIP, NodePort ve LoadBalancer servis tipleri |
| ingress.yaml | HTTP(S) yönlendirme (Klasik yöntem) |
| gateway_api.yaml | **2026 Standardı:** Rol tabanlı trafik yönetimi |
| networkpolicy.yaml | L3/L4 ağ izolasyon kuralları |
